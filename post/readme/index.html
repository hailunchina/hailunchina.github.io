关于自己

2020-09-16

我叫尹海轮,来自于江苏数千,高中未念完辍学,还未满18岁的我到社会上打工,
从最开始的饭店服务员,保安,洗车工,到现在的程序员,从月薪800左右,到后来的3~4万一月,
10年历程中,我是如何走到了现在,我用我的经验和经历告诉你,你该如何做,
少走弯路,快速达到目标,我在学习工作和工作方向的选择中,走错了很多的路线,
找错了方向,也耽误了很多时间,导致现在后期乏力,现在辞职在家,打算转行,
也正是因为这些经历,让我可以知道哪些地方需要踩,哪些地方不需要踩,
在没有人指点的情况下,我绕了很多的路,希望看到该文章的你,不用重复踩这样重复的坑.
谈谈成就,我这样的人,曾在全中国最高级别的网络中心工作过,全中国的网络,无论是天上的,还是地下的,无论是电信,移动还是铁通,最终都会由这个国家机关链接到国际互联网,保密单位,不方便透露.
我在杭州某上市公司,研发了国际首创技术,达到行业领先水平,直接带动产品的用户量大批量上涨,在公司内部报中得到了专班介绍,之后还在一些上市公司和一些非上市公司工作过,只要想做,没有什么不可能,别人能做的,你也能做到.
我会用我的经验告诉你,如何通过我的教学,在最短时间内,可以得到最大的利益化,将我的所学所知做一个总结,让更多的人可以步入这个行业,了解这个行业.
想要进入这个行业,有些人会问,我英语和数学不太好,可以做吗,嗯,是的,我也不太好,你想对了,但是其实这个行业并不是太需要英语和数学,那数学和英语重要吗,非常重要,特别是从事开发的人员,英语至关重要,数学在专业的领域上会涉及到,所以也很重要,但常规的开发过程中其实用到英语不太多,给自己定一个目标就好,从现在开始,每天记住两个行业英语单词的发音和拼写,熟练掌握两个英语单词,对很多人来说应该并不难,一年后,你就可以成功掌握了700专业英语,足可以阅读大部分的技术性英文文档,英语的体现价值往往是后期,而非前期,英语能阅读能力,代表你看英文文章的熟练程度,代表你能走多高,走多远,专业的知识目前还是以英文资料提供,使用翻译软件翻译的文章会描述的不准确,容易误导人,而且翻译后的文章和原文说表达的意思也可能完全不是一个东西,所以看原文英文是非常重要的一件事情.特别是当技术能力达到一定程度的时候,这个时候也是非常体现英文阅读能力的时候,但是告诉大家一个好消息是,专业英语其实不难,学起来也简单,单词量也不多,所以每天两个英文单词,一年后,你就可以熟悉掌握大部分英文资料.
我用10年走完的路,跟着我的教学,预估不到两年,就可以就可以达到我的专业成就,而学习完所有的技术体系,一年时间就完全够了,剩下一年从业即可,有些人教学,讲一大堆东西,却没有干货,没有让小白可以一听就明白,因为有些人起点就很高,比如大学毕业生,或者电子,计算机,通信网络等部分专业学校毕业的人,他们在学校学习的过程中有些概念是听过的,有一个印象,在加上有一定的英语能力,所以起步略快,这类学生基础好,但实践能力却完全不够,所以如果同时学习进入行业,起步和成就基本也一样.
所以所有学员,无论你是什么行业,什么水平,什么毕业,都不是关键因素,只要跟着我的教学,你就可以做到,我高中未念完辍学,英语又不会,数学都是倒数的人,一步一步走到了今天,掌握了知识,补充自己不足,我就是最好的证明,我希望看到我文章的人稍安勿躁,你能做的就是稳定的朝着一个方向,用最快的方向达到你的目标,而我就是在这个行业一个最好的领路人.
写在后面,基础好的人并不是很多,大学生在所有中国学生中也只是百分之一的比例,更多的人面对他们的是迷茫,是不知所措,是看不清未来的路在哪里,不要和一个有起点的人谈论起点的问题,那对你来说,没有帮助,无论你的什么水平,来到我这里都是从零开始,以下我谈论一下我的观点智者见智仁者见仁(不喜忽喷),有些人学习的是护校,有些人学习的是医生,护校出来只能当护士,而医生出来就是当医生,护士干几年年龄大了就被淘汰了,而医生越干越值钱,那你为啥不学医,而是学习护校呢?因为门槛,而门槛是底层大众服务上层大众的,上层资源有限,如果人人都学医,那么还有护士吗,?还有低廉的服务人员吗?所以我们要通过一切手段学医,学校去不了就去培训班,.然后考证拿到医师证,最后有了医师证,我们就可以去医院上班工作,做一个医生,如果医院不要,我们可以自己开个医院,也可以开一个诊所,走医院的路,让医院无路可走,这就是民办医院,而也许国家可能并不希望这种事情发生,因为缺少了底层的服务人员,上层人的空间就会被压缩,这个是一个很奇怪的怪圈,更多的话我就不说了,可能有些人会认为我是反社会了,回到技术上来,我们就是要学习别人能做的,然后做别人一样的工作,现在是互联网时代,知识和咨询发达,而你开发的软件或者做的产品好用,那么别人就会用,跟基础,跟能力,跟起点,都没关系,所以我的教学目标也是希望这样的人越来愈多,让更多的人看到机会,让更多的人有路可走.
